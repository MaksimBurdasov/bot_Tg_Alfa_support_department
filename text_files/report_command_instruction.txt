Как работать с командой rep?

Я использую репозиторий: https://git.moscow.alfaintra.net/projects/OSSKTB/repos/rc-telegram-bot/browse/sql_queries

Замени FILE в rep_FILE на название файла, SQL-запрос из которого хочешь выполнить. Отправь команду. Если файл с таким названием есть в репозитории, я пришлю результат Excel-файлом.

Пример: чтобы выполнить Ф.sql, отправь команду rep_Ф (не забывай слэш)

Учти, что:
 а) сервер выбирается по ID чата;
 б) в названии фала не должно быть пробелов.
